# Intro

- **简介：**Staking协议的本质来源于PoS共识需要质押一定的原生代币来参与节点的验证和出块，但对于毫无经验的小白，自建节点和维护节点+高昂的staking代币数量都会是一个阻碍。于是staking协议通过将节点的运营者operator和持有少量代币的散户群体撮合起来，operator出基建而散户则出钱，从而使得人人都能获得去中心化网络运营所带来的收入。
